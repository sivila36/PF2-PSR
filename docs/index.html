Servidor Flask - Registro de Usuarios y Gestión de Tareas

🚀 Instrucciones para ejecutar

  1. Cloná el repositorio
  2. Instalá las dependencias: 
    pip install flask
  3. Ejecutá el servidor: 
    python servidor.py
  4. Probalo desde el navegador en http://localhost:5000

📚 Conceptos clave

🔐 ¿Por qué hashear contraseñas?

Hashear contraseñas protege a los usuarios si la base de datos es comprometida. El hashing convierte la contraseña en una representación irreversible que no se puede desencriptar. Además, evita que contraseñas comunes puedan ser adivinadas fácilmente.

🗃️ ¿Por qué usar SQLite?

SQLite es ideal para proyectos pequeños o educativos como este. No requiere instalación de servidor, es liviano, y almacena la base de datos en un archivo local. Es fácil de usar y muy portable.
